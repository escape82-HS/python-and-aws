1 Variabili e Input/Output

Le variabili sono contenitori per i dati. L'Input riceve dati dall'utente, l'Output li stampa a schermo. In Python, usiamo input() e print().

🛠️ Prova Pratica 1: Il Calcolatore di Età

Crea un file chiamato pratica1.py e inserisci questo codice:

# Chiediamo i dati all'utente
nome = input("Inserisci il tuo nome: ")
anno_nascita_str = input("In che anno sei nato? ")

# Convertiamo la stringa in un numero intero (int) per fare calcoli matematici
anno_nascita = int(anno_nascita_str)
anno_corrente = 2024

# Calcoliamo l'età
eta = anno_corrente - anno_nascita

# Stampiamo il risultato usando una f-string (stringa formattata)
print(f"\nCiao {nome}! Nel {anno_corrente} compi (o hai compiuto) {eta} anni.")

▶️ Esecuzione:

$ python pratica1.py Inserisci il tuo nome: Mario In che anno sei nato? 1995 Ciao Mario! Nel 2024 compi (o hai compiuto) 29 anni.

2 Condizioni e Cicli

Le condizioni (if/elif/else) permettono al programma di prendere decisioni. I cicli (for) permettono di ripetere un'azione su una lista di elementi.

🛠️ Prova Pratica 2: Controllo Stato Server

Immaginiamo di avere una lista di server con stati diversi. Crea un file pratica2.py:

# Creiamo una lista con lo stato di 4 server
stati_server = ["running", "stopped", "running", "terminated"]

numero_server = 1

# Il ciclo for analizza un elemento alla volta dalla lista
for stato in stati_server:
  print(f"Analisi Server {numero_server}...")
  
  # Condizioni logiche
  if stato == "running":
      print("  ✅ Il server è operativo.")
  elif stato == "stopped":
      print("  ⚠️ Il server è fermo. Ricordati di accenderlo.")
  else:
      print("  ❌ Il server è stato eliminato.")
      
  numero_server += 1  # Passiamo al server successivo
  print("-" * 30) # Stampa una linea separatrice

▶️ Esecuzione:

$ python pratica2.py Analisi Server 1... ✅ Il server è operativo. ------------------------------ Analisi Server 2... ⚠️ Il server è fermo. Ricordati di accenderlo. ------------------------------ Analisi Server 3... ✅ Il server è operativo. ------------------------------ Analisi Server 4... ❌ Il server è stato eliminato. ------------------------------

3 Funzioni (Parametri in Ingresso e Uscita)

Le funzioni (definite con def) sono blocchi di codice riutilizzabili. Accettano dati in ingresso (parametri) e restituiscono un risultato in uscita (return).

🛠️ Prova Pratica 3: Convertitore di Temperatura

Crea una funzione che converte i gradi Celsius in Fahrenheit. Crea il file pratica3.py:

# Definizione della funzione
def celsius_a_fahrenheit(gradi_celsius):
  # gradi_celsius è il parametro in INGRESSO
  calcolo = (gradi_celsius * 9/5) + 32
  return calcolo  # Il risultato in USCITA

# Programma principale
print("--- Convertitore di Temperatura ---")
temp_c = float(input("Inserisci i gradi Celsius: "))

# Chiamiamo la funzione e salviamo il risultato in una variabile
temp_f = celsius_a_fahrenheit(temp_c)

print(f"{temp_c}°C corrispondono a {temp_f}°F")

▶️ Esecuzione:

$ python pratica3.py --- Convertitore di Temperatura --- Inserisci i gradi Celsius: 25 25.0°C corrispondono a 77.0°F

4 Moduli e Librerie

Python ha tantissimi moduli pronti all'uso (come boto3 per AWS). Usiamo il comando import per includerli nel nostro script.

🛠️ Prova Pratica 4: Generatore di Password

Useremo il modulo random per creare una password sicura. Crea il file pratica4.py:

import random

def genera_password(lunghezza):
  caratteri_disponibili = "abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789!@#$%"
  password = ""
  
  for i in range(lunghezza):
      # Sceglie un carattere a caso e lo aggiunge alla password
      carattere_casuale = random.choice(caratteri_disponibili)
      password += carattere_casuale
      
  return password

print("Generazione di 3 password sicure da 12 caratteri:")
for x in range(3):
  print(f"Password {x+1}: {genera_password(12)}")

▶️ Esecuzione (i risultati saranno diversi ogni volta!):

$ python pratica4.py Generazione di 3 password sicure da 12 caratteri: Password 1: aB7!x9Qz$mP2 Password 2: 9vL#kR4tY!w1 Password 3: cX5$nB8mP@qL

🏆 Sfida Finale: Simulatore di Deploy

Uniamo tutto quello che abbiamo imparato! Creeremo uno script che simula l'avvio di server cloud, utilizzando moduli, funzioni, cicli e condizioni. Crea il file sfida_finale.py:

import time
import random

def avvia_istanza_ec2(nome_istanza):
  print(f"\n[AWS Simulato] Richiesta avvio per: {nome_istanza}")
  
  # Simuliamo un tempo di attesa con un ciclo e il modulo time
  for i in range(3, 0, -1):
      print(f"Avvio in corso... {i} sec")
      time.sleep(1) # Mette in pausa lo script per 1 secondo
      
  # Simuliamo un tasso di successo dell'80%
  probabilita_successo = random.randint(1, 10)
  
  if probabilita_successo <= 8:
      return "RUNNING"
  else:
      return "FAILED"

# --- PROGRAMMA PRINCIPALE ---
miei_server = ["Web-Server-1", "Database-Primario", "Cache-Server"]

print("Inizio procedura di deployment automatico...")

for server in miei_server:
  esito = avvia_istanza_ec2(server)
  
  if esito == "RUNNING":
      print(f"✅ SUCCESSO: L'istanza {server} è online!")
  else:
      print(f"❌ ERRORE: Impossibile avviare {server}. Contattare il supporto.")

print("\nDeployment terminato.")

Esegui questo script con python sfida_finale.py. Noterai che il programma "aspetta" realmente 3 secondi per ogni server, simulando un'operazione reale. Inoltre, grazie al modulo random, a volte un server potrebbe fallire l'avvio!
