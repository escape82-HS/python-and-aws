Esercitazione 1: Setup dell'Ambiente e il tuo Primo Script

⏱️ Durata stimata: 60 minuti

In questa prima fase prepareremo la nostra postazione di lavoro, essenziale per tutto il corso, e scriveremo il nostro primo programma interattivo.

Obiettivi:

  • Installazione di Python e verifica dal terminale.
  • Creazione della cartella di lavoro del corso.
  • Utilizzo delle funzioni base print() e input().
  • Assegnazione di variabili e conversione di tipo (Casting).

Guida Passo-Passo:

  1. Prepara l'ambiente: Apri il tuo IDE (es. VS Code). Crea una nuova cartella chiamata Corso_Python_AWS sul tuo Desktop. All'interno di questa cartella, crea un nuovo file e chiamalo saluto.py. Assicurati di includere l'estensione .py!
  2. Il primo output: Alla riga 1 del tuo file, scrivi il comando per stampare un messaggio di benvenuto. Usa la funzione print("Benvenuto al corso Marblemind!"). Esegui lo script per verificare che funzioni.
  3. Chiedere informazioni (Input): Ora vogliamo che il programma ci chieda il nome. Crea una variabile chiamata nome e assegnale il valore restituito dalla funzione input(). Esempio: nome = input("Come ti chiami? ").
  4. Chiedere l'età: Fai la stessa cosa per l'età, creando una variabile eta_stringa. Ricorda che input() legge tutto come se fosse testo (stringa).
  5. Conversione del dato (Casting): Per fare calcoli matematici, l'età deve essere un numero. Crea una nuova variabile eta_numero e usa la funzione int() per convertire la stringa. Esempio: eta_numero = int(eta_stringa).
  6. Calcolo e Output finale: Calcola l'anno di nascita sottraendo eta_numero all'anno corrente (es. 2024). Infine, usa una f-string per stampare un messaggio finale: print(f"Ciao {nome}, sei nato nel {anno_nascita}").
Attenzione agli errori comuni: Controlla sempre di aver chiuso tutte le parentesi tonde () e le virgolette "". Se il terminale mostra un SyntaxError, controlla la riga indicata dall'errore!

Esercitazione 2: Calcolatore di Spese Aziendali (Variabili e I/O)

⏱️ Durata stimata: 50 minuti

Ora che sappiamo come prendere dati in input e convertirli, creiamo uno script utile per calcolare i rimborsi spesa di una trasferta aziendale.

Obiettivi:

  • Gestione di variabili numeriche decimali (Float).
  • Operazioni matematiche di base (+).
  • Formattazione avanzata delle stringhe per mostrare i decimali.

Guida Passo-Passo:

  1. Crea un nuovo file chiamato rimborsi.py nella tua cartella di lavoro.
  2. Chiedi all'utente il costo del volo usando input(). Poiché i costi possono avere i centesimi, converti subito l'input in un numero decimale usando float() invece di int(). Salva il risultato in una variabile costo_volo.
  3. Ripeti il passaggio precedente per chiedere il costo_hotel e il costo_pasti.
  4. Crea una variabile totale_spese che sia la somma delle tre variabili precedenti.
  5. Stampa un resoconto finale. Per fare in modo che i numeri mostrino sempre due cifre decimali (es. 150.50 invece di 150.5), usa questa sintassi nelle f-string: f"Volo: € {costo_volo:.2f}".
Suggerimento: Puoi concatenare le funzioni per scrivere meno codice. Invece di fare due passaggi, puoi scrivere:
costo = float(input("Inserisci costo: "))

Esercitazione 3: Il Controllo degli Accessi (If/Else e Cicli While)

⏱️ Durata stimata: 60 minuti

Introduciamo la logica condizionale e i cicli. Simuleremo un sistema di login che continua a chiedere la password finché non è corretta o finché non si esauriscono i tentativi.

Obiettivi:

  • Utilizzo del ciclo while per ripetere un'azione.
  • Costrutti condizionali if, else.
  • Gestione di un contatore.

Guida Passo-Passo:

  1. Crea un file chiamato login.py.
  2. Definisci le tue variabili iniziali:
    - password_segreta = "python2024"
    - tentativi_fatti = 0
  3. Scrivi l'intestazione del ciclo while: vogliamo che il ciclo continui mentre i tentativi fatti sono minori di 3. while tentativi_fatti < 3:
  4. Dentro il ciclo (ricorda l'indentazione!): Chiedi all'utente di inserire la password con input().
  5. Sempre dentro il ciclo, aumenta il contatore dei tentativi: tentativi_fatti = tentativi_fatti + 1.
  6. Aggiungi un blocco if: se la password inserita è uguale (==) alla password segreta, stampa "Accesso consentito" e usa il comando break per interrompere forzatamente il ciclo.
  7. Aggiungi un blocco else: se la password è sbagliata, stampa "Password errata. Riprova."
  8. Fuori dal ciclo (senza indentazione): Controlla se l'utente ha fallito. if tentativi_fatti == 3 and password_inserita != password_segreta: stampa "Account bloccato, contatta www.marblemind.eu".

Esercitazione 4: Analisi di una stringa (Cicli For)

⏱️ Durata stimata: 50 minuti

I cicli for sono fondamentali in Python per iterare (scorrere) sugli elementi di una sequenza, come i caratteri di una parola.

Obiettivi:

  • Utilizzo del ciclo for su una stringa.
  • Utilizzo dell'operatore in.
  • Metodi delle stringhe (es. .lower()).

Guida Passo-Passo:

  1. Crea un file chiamato analizzatore.py.
  2. Chiedi all'utente di inserire una frase e salvala nella variabile frase.
  3. Prepara un contatore impostandolo a zero: numero_vocali = 0.
  4. Definisci una stringa di riferimento con le vocali: vocali = "aeiou".
  5. Inizia il ciclo: for lettera in frase: (Questo significa: per ogni singola lettera presente nella frase, esegui il blocco di codice seguente).
  6. Dentro il ciclo: Per evitare problemi con le maiuscole, trasforma la lettera in minuscolo prima di controllarla: lettera_min = lettera.lower().
  7. Usa un if per controllare se la lettera minuscola è presente nella stringa delle vocali: if lettera_min in vocali:
  8. Se è vero, incrementa il contatore: numero_vocali += 1.
  9. Fuori dal ciclo: Stampa il risultato finale mostrando la frase originale e quante vocali contiene.
