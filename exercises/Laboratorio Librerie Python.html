🧠 Teoria: Cos'è una Libreria?

Immagina Python come una cassetta degli attrezzi base. Quando lo installi, hai a disposizione martello, cacciavite e pinze (funzioni come print(), input(), len()). Ma cosa succede se devi fare un lavoro specifico, come l'idraulico o l'elettricista?

Invece di costruire nuovi attrezzi da zero, vai in un negozio e compri un "kit specifico". In Python, questi kit si chiamano Librerie (o Moduli).

Esistono due tipi di librerie:

  1. Librerie Standard (Built-in): Sono già incluse in Python quando lo installi, ma sono "nascoste" per non appesantire il programma. Per usarle basta usare il comando import (es. import math).
  2. Librerie Esterne (Third-party): Create dalla community (come Boto3 per AWS o Pandas per i dati). Non sono presenti sul tuo PC. Prima di importarle, devi scaricarle e installarle da internet usando un programma chiamato PIP (Python Package Installer).

Come si installa una libreria esterna?
Si apre il Terminale (non il codice Python!) e si digita: pip install nome_libreria.
(Nota: Se usi Google Colab, devi aggiungere un punto esclamativo all'inizio: !pip install nome_libreria).

Esercitazione 1: La Libreria Standard (Math e Random)

⏱️ Durata stimata: 15 minuti

Iniziamo utilizzando due librerie già presenti nel tuo computer. Creeremo un piccolo script che genera un numero casuale e ne calcola la radice quadrata.

Guida Passo-Passo:

  1. Importazione: Crea un nuovo file chiamato matematica.py (o una nuova cella su Colab). Alla primissima riga, importa le due librerie necessarie: 
    import random
import math
  2. Generare un numero casuale: Usa la funzione randint(min, max) che si trova dentro la libreria random. Per usarla, devi scrivere nome_libreria.nome_funzione. 
    numero_segreto = random.randint(1, 100)
print(f"Il numero casuale generato è: {numero_segreto}")
  3. Calcolare la radice quadrata: Ora usiamo la libreria math e la sua funzione sqrt() (Square Root). 
    radice = math.sqrt(numero_segreto)
print(f"La radice quadrata di {numero_segreto} è {radice:.2f}")
  4. Esecuzione: Esegui lo script più volte. Noterai che il numero cambia ad ogni esecuzione, dimostrando che il modulo random sta facendo il suo lavoro!

Esercitazione 2: Lavorare con le Date (Datetime)

⏱️ Durata stimata: 20 minuti

Gestire il tempo (giorni, ore, fusi orari) in programmazione è difficilissimo se fatto da zero. Fortunatamente, Python ha la libreria standard datetime. Qui impareremo un modo diverso di importare: importare solo uno strumento specifico dalla cassetta.

Guida Passo-Passo:

  1. Importazione Specifica: Invece di importare tutta la libreria, importiamo solo l'oggetto "datetime" e l'oggetto "timedelta" (che serve per fare calcoli coi giorni). 
    from datetime import datetime, timedelta
  2. Ottenere l'ora attuale: Usa la funzione now(). 
    oggi = datetime.now()
print(f"Data e ora grezza: {oggi}")
  3. Formattare la data: Il formato "grezzo" è brutto da leggere. Usiamo il metodo strftime (String Format Time) per renderlo leggibile per un utente italiano (Giorno/Mese/Anno). 
    data_formattata = oggi.strftime("%d/%m/%Y %H:%M:%S")
print(f"Data formattata: {data_formattata}")
  4. Viaggiare nel tempo: Vogliamo sapere che giorno sarà tra esattamente 30 giorni. Usiamo timedelta. 
    scadenza = oggi + timedelta(days=30)
print(f"La tua iscrizione scadrà il: {scadenza.strftime('%d/%m/%Y')}")
Nota bene: Usando from ... import ... non hai più bisogno di scrivere il nome della libreria prima della funzione. Hai importato l'attrezzo direttamente sul tuo banco di lavoro!

Esercitazione 3: Installare una Libreria Esterna (Requests)

⏱️ Durata stimata: 25 minuti

Ora facciamo sul serio. Vogliamo che il nostro script Python si colleghi a un sito web e scarichi dei dati. Per farlo ci serve la libreria requests, che non è inclusa in Python. Dobbiamo installarla tramite PIP.

Guida Passo-Passo:

  1. Installazione tramite PIP:
    👉 Se sei sul tuo PC (VS Code/Terminale): Apri il terminale e digita pip install requests e premi Invio.
    👉 Se sei su Google Colab: Crea una cella di codice, scrivi !pip install requests ed eseguila.
  2. Importazione: Ora che è installata, possiamo importarla nel nostro script. 
    import requests
import json
  3. Effettuare una richiesta Web (GET): Interrogheremo un'API pubblica e gratuita che restituisce una barzelletta (in inglese) in formato JSON. 
    url = "https://official-joke-api.appspot.com/random_joke"
print("Mi sto connettendo al server...")

# Effettuiamo la richiesta
risposta = requests.get(url)
  4. Controllo e Lettura dei Dati: Verifichiamo che la richiesta sia andata a buon fine (Status Code 200 = OK) e stampiamo il risultato. 
    if risposta.status_code == 200:
  # Convertiamo la risposta testuale in un dizionario Python
  dati = risposta.json()
  
  print("\n--- BARZELLETTA DEL GIORNO ---")
  print(f"Domanda: {dati['setup']}")
  print(f"Risposta: {dati['punchline']}")
  print("------------------------------")
else:
  print(f"Errore di connessione. Codice: {risposta.status_code}")
  5. Esecuzione: Lancia lo script! Hai appena creato un programma che comunica con un server esterno su internet. Questo è l'esatto principio con cui, nel prossimo modulo, comunicheremo con i server di AWS usando Boto3!
