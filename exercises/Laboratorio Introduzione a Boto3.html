Laboratorio: Primi Passi con Boto3

Materiale didattico a cura di www.marblemind.eu

Obiettivo: Comprendere l'SDK AWS per Python, configurare le credenziali e interagire con EC2 e S3.

🧠 Concetti Base: Cos'è Boto3?

Boto3 è l'SDK (Software Development Kit) ufficiale di Amazon Web Services per Python. Permette di creare, configurare e gestire i servizi AWS (come EC2 e S3) scrivendo codice Python invece di cliccare sulla console web.

Boto3 offre due modi principali per interagire con AWS:

  • Client: Un'interfaccia a basso livello. Restituisce le risposte esatte delle API AWS sotto forma di grossi dizionari Python. È completo ma richiede più codice per estrarre i dati.
  • Resource: Un'interfaccia ad alto livello (Object-Oriented). Restituisce oggetti Python facili da manipolare (es. istanza.id o istanza.state). È più semplice da usare per i principianti.

Esercitazione 1: Setup e Lettura Risorse (EC2)

⏱️ Durata stimata: 30 minuti

Iniziamo configurando l'ambiente e scrivendo uno script di sola lettura (Read-Only) per interrogare il nostro account AWS e scoprire quali istanze EC2 sono attualmente in esecuzione.

Guida Passo-Passo:

  1. Installazione: Apri il terminale del tuo IDE e installa la libreria digitando: pip install boto3.
  2. Configurazione Credenziali: Boto3 ha bisogno di sapere chi sei per accedere ad AWS. Se hai già installato la AWS CLI, digita aws configure nel terminale e inserisci la tua Access Key, Secret Key e la regione di default (es. eu-west-1).
  3. Il primo script: Crea un file chiamato lista_ec2.py. Inizia importando la libreria: 
    import boto3
  4. Creazione della Resource: Inizializza l'interfaccia ad alto livello per il servizio EC2. 
    ec2 = boto3.resource('ec2')
print("Connessione ad AWS stabilita con successo!")
  5. Interrogazione (Listing): Usa un ciclo for per scorrere tutte le istanze presenti nel tuo account e stampare il loro ID e lo stato attuale (es. running, stopped). 
    print("\n--- Elenco Istanze EC2 ---")
# ec2.instances.all() recupera tutte le istanze
for istanza in ec2.instances.all():
  id_istanza = istanza.id
  stato = istanza.state['Name']
  tipo = istanza.instance_type
  
  print(f"ID: {id_istanza} | Tipo: {tipo} | Stato: {stato}")
  6. Esecuzione: Salva il file ed eseguilo. Se ricevi un errore NoCredentialsError, significa che il passaggio 2 non è andato a buon fine. Chiedi supporto al docente!

Esercitazione 2: Gestione Storage (Amazon S3)

⏱️ Durata stimata: 40 minuti

Ora passeremo all'azione creando risorse. Utilizzeremo Boto3 per creare un nuovo Bucket S3 e caricarci dentro un file di testo generato dal nostro script Python.

Regola d'oro di S3: Il nome di un Bucket S3 deve essere univoco a livello globale in tutto il mondo AWS. Non puoi chiamarlo "test-bucket". Usa un prefisso unico, ad esempio il tuo nome e la data di oggi (es. marblemind-mario-20240512).

Guida Passo-Passo:

  1. Preparazione: Crea un nuovo file chiamato gestione_s3.py.
  2. Inizializzazione Client: Questa volta useremo il client per S3. 
    import boto3

# Inizializziamo il client S3
s3_client = boto3.client('s3')

# Definisci il nome del tuo bucket (CAMBIALO CON UNO UNIVOCO!)
nome_bucket = "marblemind-tuonome-12345"
  3. Creazione del Bucket: Aggiungi il codice per creare il bucket. Usiamo un blocco try-except per gestire l'errore nel caso in cui il nome scelto esista già. 
    try:
  print(f"Tentativo di creazione del bucket: {nome_bucket}...")
  # Nota: se usi una regione diversa da us-east-1, serve specificare la LocationConstraint
  s3_client.create_bucket(
      Bucket=nome_bucket,
      CreateBucketConfiguration={
          'LocationConstraint': 'eu-west-1' # Cambia con la tua regione
      }
  )
  print("Bucket creato con successo!")
except Exception as e:
  print(f"Errore durante la creazione: {e}")
  4. Creazione di un file locale: Facciamo in modo che Python crei un semplice file di testo sul nostro computer. 
    nome_file = "saluto_aws.txt"

# Creiamo un file e ci scriviamo dentro
with open(nome_file, "w") as file:
  file.write("Ciao AWS! Questo file è stato caricato tramite Python e Boto3 durante il corso Marblemind.")
  
print(f"File locale '{nome_file}' creato.")
  5. Upload su S3: Ora usiamo il metodo upload_file per inviare il file appena creato al nostro nuovo bucket. 
    print("Inizio caricamento su S3...")
s3_client.upload_file(
  Filename=nome_file,      # Il file sul tuo PC
  Bucket=nome_bucket,      # Il bucket di destinazione
  Key="documenti/saluto.txt" # Il nome e percorso che avrà su S3
)
print("Caricamento completato! Controlla la console AWS.")
  6. Verifica: Esegui lo script. Vai sulla console web di AWS, apri il servizio S3, cerca il tuo bucket e verifica che all'interno ci sia una cartella "documenti" contenente il tuo file!
