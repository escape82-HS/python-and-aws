☁️ Lab: Deploy di un'istanza AWS EC2 con Python

Una guida passo-passo per principianti per automatizzare la creazione di server cloud utilizzando Python e la libreria Boto3.

📋 Prerequisiti

  • Un Account AWS attivo.
  • Python 3 installato sul tuo computer.
  • Un utente IAM su AWS con permessi di accesso programmatico (Access Key ID e Secret Access Key) e permessi per creare istanze EC2 (es. AmazonEC2FullAccess).

1 Preparazione dell'Ambiente

Per far comunicare Python con AWS, utilizzeremo la libreria ufficiale chiamata Boto3. Inoltre, useremo la AWS CLI per configurare in modo sicuro le nostre credenziali.

1. Installa le librerie necessarie:

Apri il terminale (o Prompt dei comandi) ed esegui:

pip install boto3 awscli

2. Configura le credenziali AWS:

Nel terminale, digita il seguente comando e inserisci le tue chiavi quando richieste. Usa us-east-1 come regione di default e json come formato di output.

aws configure

Nota di sicurezza: Non inserire mai le tue chiavi AWS direttamente nel codice Python. Utilizzare `aws configure` le salva in modo sicuro nel tuo computer.

2 Scrivere il Codice Python

Crea un nuovo file chiamato deploy_ec2.py e incolla il seguente codice. Questo script creerà un'istanza di tipo t2.micro (inclusa nel piano gratuito di AWS).

import boto3

# 1. Inizializza la connessione al servizio EC2 nella regione specificata
ec2 = boto3.resource('ec2', region_name='us-east-1')

print("Richiesta di creazione dell'istanza inviata. Attendere...")

# 2. Crea l'istanza
instances = ec2.create_instances(
  ImageId='ami-0c7217cdde317cfec', # AMI di Ubuntu 22.04 in us-east-1
  MinCount=1,                      # Numero minimo di istanze da creare
  MaxCount=1,                      # Numero massimo di istanze da creare
  InstanceType='t2.micro',         # Tipo di istanza (Free Tier eligible)
  TagSpecifications=[              # Diamo un nome alla nostra istanza
      {
          'ResourceType': 'instance',
          'Tags': [
              {'Key': 'Name', 'Value': 'LaMiaPrimaIstanzaPython'}
          ]
      }
  ]
)

# 3. Ottieni l'oggetto dell'istanza appena creata
instance = instances[0]

# 4. Attendi che l'istanza sia completamente avviata ("running")
print("Attesa dell'avvio dell'istanza...")
instance.wait_until_running()

# 5. Aggiorna i dati dell'istanza per ottenere l'Indirizzo IP pubblico
instance.reload()

print("\n✅ Istanza creata con successo!")
print(f"ID Istanza: {instance.id}")
print(f"Indirizzo IP Pubblico: {instance.public_ip_address}")

Cos'è l'ImageId (AMI)? È il sistema operativo base. L'ID ami-0c7217cdde317cfec corrisponde a Ubuntu Linux nella regione us-east-1. Se usi una regione diversa, dovrai cercare l'ID AMI corrispondente nella console AWS.

3 Esecuzione e Verifica

Ora sei pronto per lanciare il tuo server nel cloud!

  1. Apri il terminale nella cartella dove hai salvato il file.
  2. Esegui lo script con il comando: python deploy_ec2.py
  3. Attendi qualche secondo. Vedrai stampato a schermo l'ID della tua nuova istanza e il suo indirizzo IP pubblico.
  4. Vai sulla Console AWS, cerca il servizio EC2, clicca su Instances e vedrai il tuo server in esecuzione!

4 Pulizia (MOLTO IMPORTANTE)

Per evitare che AWS ti addebiti dei costi, devi sempre spegnere ed eliminare (Terminare) le risorse che non usi più.

Crea un file termina_ec2.py ed esegui questo codice, sostituendo l'ID con quello che hai ottenuto nello step precedente:

import boto3

ec2 = boto3.resource('ec2', region_name='us-east-1')

# INSERISCI QUI L'ID DELLA TUA ISTANZA (es. 'i-0abcd1234efgh5678')
instance_id = 'INSERISCI_IL_TUO_INSTANCE_ID'

# Trova l'istanza e terminala
instance = ec2.Instance(instance_id)
instance.terminate()

print(f"Richiesta di terminazione inviata per l'istanza {instance_id}.")

Verifica sempre sulla console AWS che l'istanza sia nello stato "Terminated".

Congratulazioni! Hai appena automatizzato il cloud con Python. 🎉
