LAB 3 – Primi passi con Boto3

Guida step-by-step per connettersi ad AWS, gestire EC2 e S3 interamente con Python

0 – Checklist preparazione ambiente

  • Account AWS (utente IAM con AdministratorAccess oppure policy custom: EC2FullAccess + S3FullAccess)
  • Python ≥ 3.8 installato (preferibilmente via Homebrew su macOS/Linux o python.org su Windows)
  • AWS CLI già configurato con aws configure (region, access key, secret key, output = json)
  • Editor/IDE a scelta (VS Code, PyCharm, ecc.)

1 – Installazione e configurazione di Boto3

Crea/attiva un virtual-env (best-practice) → evita conflitti con pacchetti di sistema.

$ python3 -m venv .venv
$ source .venv/bin/activate        # Windows: .venv\Scripts\activate
(.venv) $ pip install -U pip boto3

Verifica: apri REPL Python e digita import boto3, botocore – nessun errore = OK

2 – Prima connessione: verifica identità e regione

# connessione_minima.py
import boto3, json

session = boto3.Session()          # usa credenziali e regione di aws configure
sts     = session.client('sts')
print("Account:", sts.get_caller_identity()["Account"])
print("UserArn:", sts.get_caller_identity()["Arn"])

# ECHO della region di default
print("Region   :", session.region_name)

Se ottieni Unable to locate credentials → lancia aws configure o imposta variabili AWS_ACCESS_KEY_ID e AWS_SECRET_ACCESS_KEY

3 – Listing di risorse EC2 e S3

3-A EC2 – elenca istanze attive

# list_ec2.py
import boto3

ec2 = boto3.client('ec2')

response = ec2.describe_instances()
for res in response['Reservations']:
    for inst in res['Instances']:
        state = inst['State']['Name']
        tag_name = next((t['Value'] for t in inst.get('Tags', []) if t['Key']=='Name'), '-')
        print(f"{inst['InstanceId']}  {state:>10}  {tag_name}")

3-B S3 – elenca bucket e oggetti

# list_s3.py
import boto3

s3 = boto3.client('s3')

print("Bucket list:")
for b in s3.list_buckets()['Buckets']:
    print(" -", b['Name'], "creato il", b['CreationDate'])

# Lista oggetti del primo bucket (max 10)
if s3.list_buckets()['Buckets']:
    bucket = s3.list_buckets()['Buckets'][0]['Name']
    print(f"\nPrimi 10 oggetti di {bucket}:")
    for obj in s3.list_objects_v2(Bucket=bucket, MaxKeys=10)['Contents']:
        print("   ", obj['Key'], obj['Size'], "bytes")

4 – Creare e gestire bucket S3

4-A Creazione bucket (nome univoco GLOBALE!)

import boto3, uuid

s3 = boto3.client('s3')
bucket = f"lab3-bucket-{uuid.uuid4()}"   # suffisso casuale
s3.create_bucket(
    Bucket=bucket,
    CreateBucketConfiguration={'LocationConstraint': 'eu-west-1'}
)
print(f"Creato bucket: {bucket}")

Se ricevi BucketAlreadyExists → il prefisso non è univoco, cambialo.

4-B Cancellazione bucket vuoto

s3.delete_bucket(Bucket=bucket)

4-C Cancellazione bucket NON vuoto (nuova API)

import boto3
s3 = boto3.resource('s3')          # uso resource per comodità
bucket = s3.Bucket(bucket)
bucket.objects.all().delete()    # svuota
bucket.delete()                    # rimuovi

5 – Upload / Download di file

5-A Upload: versione semplice

s3.upload_file(Filename='report.pdf', Bucket=bucket, Key='docs/report.pdf')
print("Caricamento completato")

5-B Upload con metadata e ACL pubblica (es. immagini)

s3.upload_file(
    Filename='logo.png',
    Bucket=bucket,
    Key='public/logo.png',
    ExtraArgs={
        'ACL': 'public-read',
        'ContentType': 'image/png',
        'Metadata': {'author': 'lab3', 'project': 'demo'}
    }
)

Link pubblico: https://[bucket].s3.eu-west-1.amazonaws.com/public/logo.png

5-C Download

s3.download_file(Bucket=bucket, Key='docs/report.pdf', Filename='report_local.pdf')

5-D Upload/Download in RAM (oggetti piccoli)

# stringa → S3
s3.put_object(Bucket=bucket, Key='hello.txt', Body=b'Ciao da LAB 3')

# S3 → stringa
response = s3.get_object(Bucket=bucket, Key='hello.txt')
print(response['Body'].read().decode())   # "Ciao da LAB 3"

6 – Gestione di metadata e tags

6-A Metadata (key-value visibili in Console → Proprietà → Metadata)

s3.copy_object(
    Bucket=bucket,
    CopySource=f'{bucket}/public/logo.png',
    Key='public/logo_v2.png',
    Metadata={'checked': 'true', 'resize': '256x256'},
    MetadataDirective='REPLACE'   # importante: sovrascrivi
)

6-B Tag (massimo 10, usati per policy/filtraggio)

s3.put_object_tagging(
    Bucket=bucket,
    Key='public/logo.png',
    Tagging={
        'TagSet': [
            {'Key': 'Env',   'Value': 'lab'},
            {'Key': 'Owner', 'Value': 'student'}
        ]
    }
)

Leggere i tag:

tags = s3.get_object_tagging(Bucket=bucket, Key='public/logo.png')['TagSet']
print(tags)

7 – Esercizi da consegnare

  1. EC2 inventory: salva su CSV id, tipo, stato, nome-tag di TUTTE le istanze presenti (usa csv.DictWriter).
  2. S3 backup script: sincronizza una cartella locale ./data in un bucket (overwrite solo se file più recente). Suggerimento: confronta LastModified dell’oggetto S3 con os.path.getmtime().
  3. Tagging policy: scrivi funzione che aggiunge tag Project=lab3 a TUTTI i bucket creati oggi; se ne trova senza tag, li aggiunge a posteriori.
  4. Ec2 power-scheduler (opzionale): spegni le istanze con tag OfficeHours=false ogni venerdì sera (usa ec2.stop_instances()).
